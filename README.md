# Tech stack & Open-source libraries

* **100% Kotlin based**.
* **MVVM Architecture (View - ViewModel - Model) together with Events that decide what Fragment or Activity should do**.
* **Room Database- A Persistence library that provides an abstraction layer over SQLite database**.
* **Kotlin Coroutines to control Room operations and Threading**.
* **Material Design Components**.
